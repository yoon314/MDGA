# 🏫 한국디미고 입학 성적 산출 프로그램

> **예비 디미고 1학년 학생들을 위한 입학 전형 총점 계산기입니다.** > 일반전형과 특별전형(대회입상, 자기추천)을 선택하여 교과 성적, 생기부 점수, 자기소개서, 실적 점수 등을 합산한 최종 점수를 미리 산출해 볼 수 있습니다.

---

## 🚀 주요 기능

* **전형 선택 기능**: 일반전형과 특별전형 중 본인이 지원하고자 하는 전형을 선택할 수 있습니다.
* **성적 입력 및 예외 처리**: `dimigo.h`에 정의된 함수들을 통해 교과 및 생기부 성적을 입력받으며, 잘못된 입력 시 해당 단계로 돌아가는 예외 처리(`goto` 문 활용)가 포함되어 있습니다.
* **전형별 자동 점수 가중치 계산**:
    * **일반전형**: 교과 점수 + 생기부 점수 (만점: 140점)
    * **특별전형**: (교과 점수 / 2) + 생기부 점수 + 자기소개서 + 실적/자기추천 점수 (만점: 160점)
* **깔끔한 CLI 인터페이스**: 각 단계마다 화면을 초기화(`system("clear")`)하여 쾌적한 텍스트 기반 UI를 제공합니다.

---

## 🛠 실행 흐

1.  **시작 화면**: 디미고 로고 ASCII 아트와 함께 시작 (`Enter` 입력)
2.  **전형 선택**: 일반전형(1) 또는 특별전형(2) 선택
3.  **기본 점수 입력**:
    * 교과 점수 입력 (`subject_grade`)
    * 학생생활기록부 점수 입력 (`seng_gi_bu`)
4.  **특별전형 추가 입력 (특별전형 선택 시)**:
    * 세부 전형 선택 (1. 대회입상 / 2. 자기추천)
    * 자기소개서 점수 입력 (`cover_letter`)
    * 대회 실적 혹은 자기추천 실적 점수 입력
5.  **최종 결과 출력**: 전형별 총점(140점 혹은 160점) 대비 본인의 산출 점수를 소수점 셋째 자리까지 출력 후 종료

---

## 💻 파일 구조

* `main.c`: 프로그램의 전체적인 실행 흐름 제어 및 점수 합산, 메인 UI 출력
* `dimigo.h`: 성적 입력, 전형 선택 등 세부 기능을 수행하는 함수들의 선언 및 구현체 (헤더 파일)

---
